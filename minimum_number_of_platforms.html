Minimum Number of Platform Needed

The arrival and departure time of all trains that reach a railway station is given. Find the minimum number of platforms required for the railway station.

Arrival times: 9.00, 9.40, 9.50, 11.00
Departure times: 9.10, 11.20, 11.30, 12.00

Algorithm:
We’ll apply sorting here.

Steps:
          1. Take two arrays of arrival times and departure times.
         2. At first, sort all the arrival and departure times in ascending order.
         3. Then assign two pointers on the first element of two arrays.
         4. Assign two variables like number of platform exists (p=1) and number of platform needed (n=0).
         5. Now, check if the element array of arrival is greater than the element of departure array.
         6. If it’s greater than value of n will increase.
         7. Else the value of n will decrease.
         8. And every time the departure happens before arrival, move the pointer forward.
         9. After all the iterations check if n > p and if it’s true update the value of p to n.
