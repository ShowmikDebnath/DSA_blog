Array

An array is a Linear data structure which is a collection of items having similar data types stored in contiguous memory locations.
Or, An array is a data structure containing a number of data values (all of which are of the same type).

There are two types of array: 

                                   1. Dimensional Array (1D )
                                   2. Multi-Dimensional Array
 
                                                         (i) Two  Dimensional Array (2D)
                                                        (ii) Three Dimensional Array (3D)
