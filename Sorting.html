Sorting Algorithm

Sorting Algorithm is used to rearrange a given array or list elements according to a comparison operator on the elements. The comparison operator is used to decide the new order of element in the respective data structure.
There two kinds of sorting :
          1. Ascending order
          2. Descending order.

Ascending order : Ascending order means the process of arranging numbers from smallest to largest from left to right.
Descending order : Descending order means the process of arranging numbers from largest to smallest from right to left .

Sorting in Ascending order:
Steps:
         1. Assign two pointers on first and second element of the array.
         2. Check if the left element is smaller than the right.
         3. If it isn't smaller than, swap their position.
         4. Repeat the procedure until it is arranged as the smaller to larger from left to right.
